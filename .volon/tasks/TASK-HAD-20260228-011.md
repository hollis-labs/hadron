## Description

Implement the shared runtime configuration package used by both `hadrond` and the `hadron` CLI.

## Instructions

Create `hadron/internal/config/config.go`:

```go
type Config struct {
    Addr        string  // daemon listen address, default "127.0.0.1:8095"
    DBPath      string  // SQLite path, default ~/.hadron/state/hadron.db
    LogsDir     string  // run event logs dir, default ~/.hadron/logs/runs
    DataDir     string  // settings + misc, default ~/.hadron/
    WorkspaceID string  // default workspace, default "default"
}

func Default() *Config
func (c *Config) Ensure() error   // creates dirs if missing
```

- `Default()` resolves paths from `os.UserHomeDir()` + `/.hadron/`
- `Ensure()` runs `os.MkdirAll` on all dirs, returns first error
- Export a `DefaultAddr = "127.0.0.1:8095"` constant

## Acceptance

- `go build ./internal/config/...` passes
- `Default()` returns non-empty paths rooted in the user's home dir
- `Ensure()` creates directories if they don't exist
